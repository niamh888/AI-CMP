# AI-CMP: Compliance Review for AI-Enabled Medical Devices

This user-friently web-based agentic AI application provides a change management assessment with built-in compliance to appropriate regulatory requirements for AI-enabled medical devices.  This is based on industry and regulatory validation of content  questions extracted from a word/pdf document.

## Features

- Pre-loaded validated questions are provided in an Agentic application to support a comprehensive assessment of changes to AI-enabled Medical Device (AIeMD) products based on AI CMP Assessment document that is in compliance with regulatory and standards requirements.
- Each section includes questions with text areas for responses.  The hardcopy uploaded to the agent is ~57 pages long due to additional guidance text.  When available as a html file, it is shortened to 9 pages only.  The AI agent takes the burden out of answering the comprehensive questions.
- The user has the option of adding in ISO 27001 certification-related questions on request.
- An AI Agent will populate the responses in so far as possible based on the proposed change and details uploaded to the application.
- The AI agent will assess whether certain sections can be n/a and will allow you (the user) the opportunity to override the n/a and make it a required section.
- When the AI Agent has analyzed the content, you will be able to edit the text and add/delete as you deem necessary.  
- You will be able to download as txt, docx, or pdf file and will have a full change analysis of your proposed change.  
- You / or designated person will have admin access "admin123" to change the questions asked in the template.

## How to Use

1. Open `index.html` in a web browser.
2. Upload relevant supporting documents (pdf or word files)
3. Describe the proposed change in as much detail as possible including providing details of the requestor and source of the change and the territories impacted.
4. Click on "run assessment".
5. Accept, edit or reject the proposed text generated by the AI agent.
6. Accept, edit or reject the proposed sections as being applicable.
7. Complete the assessment and download for use in your file/eQMS or request API integration to your file system for seamless change management to the current regulatory State of the Art.

## Requirements

- A modern web browser with JavaScript enabled.
